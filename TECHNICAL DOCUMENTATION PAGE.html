Introduction

Go lang is a statically typed compiled programming language created by Google . The initial release of this programming language was at 2012 and the final release was released at the year 2021. It was created by Robert Griesemar, Rob Pike and Ken Thompson. Go is not an object oriented programming language. The advantages of Go Programming language are listed below:

  1. Go is really simple langauge to understand. It allows new programmer to pick upi the langauge quickly, and it allows experienced programmers to quickly understand someone else's code.
  2. Go is the one of the fast programming language comparitively. If you're coming from an interpreted language like PHP, Python, or Ruby, it's almost a night and day. But Go can complete those tasks in a pretty quick way

Installation and Go Commands

Installtion of Go is very simple is other programming language. The installation steps is given below:

  1. Go to the link https://golang.org/dl/ to install Go in your machine. After installtion, to check whether the installtion is coorectly done, type the following command in the command prompt go . If you get a long healthy message, then you can ensure that you have installed Go correctly on your machine.
  2. Install any of the code editors you wish, it is not mandatory to use a particular thing. But my wish is to install VS Code for better performance.
Some of the Go Commands to illustrate Go are given below:
  • go build -- Compile a bunch of go Source file.
  • go run -- Compile and execute a bunch of files.
  • go fmt -- Formats all the code in each file in the cuurent directory
  • go install -- Compiles and install a package
  • go get -- Downlaod the Raw source code of someone else's package

First Program

Now let us write our first program in Go. It's very simple as we already know. Let us write a Hello World Program
  package main
  import "fmt"
  func main(){
  fmt.Println("Hello World")
  }
The above program will execute the statement Hello World to the console.The explanation of the code is given below:

  • package main -- It's an executable type of package which contains all the subpackages. This type of package produces an executable file to the directory when you compile it.
  • import "fmt" -- It's a subpackage which contains I/O functions. fmt stands for format.
  • func main() -- It's a main function. func is a keyword to create a function in Go Programming language.
  • fmt.Println() -- It's a print function to print the neccesary statements.

Variable Declaration

There are two ways of creating variable in Go language.

  1. Using the syntax given below:
      var name_of_the_variable data_type = value
  2. Second way of decalring variable is given below:
      variable_name := value

Note that := operator is used only to create a new variable. If you want to change the value of the variable declared, you should use = operator.

Function and Return

The syntax for declaring a Function is give below:
  func name_of_the_function data_type (args passed){
  body_of_the_function

If you don't declare the datatype in the function declaration, it will throw you an error. The datatype should match the return statement what you're going to give
